### v2.2.1 (2026-06-20)

**Core Features:**
- Export selected webpage content to Word documents with one click.
- Preserve common webpage structures including paragraphs, tables, images, and hyperlinks.
- Support standard document formatting such as title/body fonts, font sizes, indentation, line spacing, margins, and page numbers.
- Generate table of contents and configure TOC styles separately.
- Support smart file naming with date, page title, department, category, and custom tags.

**Batch Export:**
- Store multiple webpage selections temporarily.
- Reorder stored items by drag-and-drop.
- Export multiple articles with unified Word document formatting.

**Settings & Usability:**
- Site blacklist for disabling the floating toolbar on selected websites.
- Configurable auto storage, toolbar buttons, and shortcut behavior.
- Chinese and English interface support.

**Privacy:**
- No account login, analytics SDK, advertising code, or tracking scripts.
- Selected content and settings are stored locally in the browser.
- No remote upload logic is included in the project code.

- 支持标题/正文字体、字号、缩进、行距、页边距、页码等标准文档格式设置。

- 支持多篇网页内容暂存。

- 支持以统一版式批量导出多篇文章。

- 支持网站禁用名单，可在指定网站关闭悬浮工具栏。
