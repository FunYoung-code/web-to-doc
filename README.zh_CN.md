## 主要功能

- **一、极简操作，轻松上手**：集成格式设置、内容管理、文档导出三大按钮，布局清晰、一键触发。
- **二、标准版式，智能适配**：支持标题、正文、页边距、缩进、行距、页码等常用文档格式设置。
- **表格绘制精准化：** 智能识别网页表格，并在 Word 中还原为表格结构。
- **图片处理批量化：** 抓取网页正文图片，自动适配页面尺寸。
- **链接文本灵活化：** 可识别超链接文本，按需选择是否留存链接格式。
- **目录生成自动化：** 支持生成文档目录，并可单独设置目录字体样式。
- **文件命名智能化：** 支持日期、标题等标签自动组合命名，可自定义部门、文件类别等标识。
- **三、批量处理，提升效能**：支持多篇网页内容统一收纳、拖拽调整排序，并批量导出。
- **四、自主设置，效率加倍**：可配置网站禁用名单、自动暂存、操作快捷键等个性化参数。

## 离线安装

1. 下载 Release 中的 ZIP 压缩包。
2. 解压到本地文件夹。
3. 打开浏览器“扩展程序”页面，并开启“开发者模式”。
4. 选择“加载已解压的扩展程序”，选中解压后的文件夹。

Chromium 浏览器可打开 `chrome://extensions/` 加载；Firefox 可在 `about:debugging#/runtime/this-firefox` 中临时加载 `manifest.json`。

## 隐私

Web to Doc 是开源扩展，代码托管在 GitHub，可公开审查。

- Web to Doc **不包含**账号登录、统计 SDK、追踪脚本或广告代码。
- Web to Doc **不会**把选中的网页内容上传到远程服务器。
- 选中的文本、表格、图片、导出设置、语言偏好、网站禁用名单等数据保存在浏览器本地。
- 为将网页图片写入 Word 文档，扩展可能会请求当前网页中的图片资源，并在本地转换。

## 权限说明

- `storage`：保存导出设置、暂存内容和网站禁用名单。
- `tabs`：获取当前标签页信息，用于识别当前网站。
- `<all_urls>`：在网页中显示悬浮工具栏，并读取用户选中的网页内容。
