# Privacy Policy

Web to Doc - 网页导出助手 runs locally in the browser.

The extension reads selected webpage content only when you use actions such as temporary storage, copy, or export. The selected text, tables, images, export settings, language preference, and site blacklist are stored in the browser's local storage.

This project does not include account login, analytics SDKs, tracking scripts, or remote upload logic. To include webpage images in exported Word documents, the extension may request image resources from the current webpage and convert them locally.

Uninstalling the extension or clearing extension storage removes locally stored data.
